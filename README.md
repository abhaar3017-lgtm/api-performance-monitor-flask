# API Performance Monitor 🚀

Real-time API monitoring dashboard built with Flask + SQLite.

## Features
- Monitor multiple APIs simultaneously
- Real-time response time graphs 
- Automatic uptime/downtime tracking
- SQLite database for historical data
- Clean, responsive UI

## Tech Stack
Python, Flask, SQLite, HTML, CSS, JavaScript, Chart.js

## How to Run
1. `pip install flask requests`
2. `python app.py`
3. Open `http://localhost:5000`
